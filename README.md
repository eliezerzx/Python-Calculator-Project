# 🧮 Python Calculator – Calculadora Interativa em Python

Este é um projeto de **calculadora interativa em Python**, desenvolvido como exercício prático para aprimorar lógica de programação, estruturas condicionais e manipulação de dados.

O programa funciona via terminal (CLI) e permite realizar diversas operações matemáticas básicas e intermediárias.

---

## 🚀 Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão (com tratamento de divisão por zero)
- Potência
- Raiz quadrada
- Porcentagem
- Menu interativo em loop
- Formatação de resultados com duas casas decimais

---

## 🛠️ Tecnologias Utilizadas

- Python 3
- Biblioteca `math`

---

📚 O que foi praticado neste projeto

Estruturas condicionais (if, elif, else)

Laços de repetição (while)

Entrada e saída de dados (input, print)

Conversão de tipos (float, int)

Funções matemáticas (math.sqrt)

Tratamento de erros (divisão por zero)

Boas práticas de organização de código
